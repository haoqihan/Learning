深浅拷贝

三种拷贝

第一种:赋值运算s1 = s2,那么他们两个指向的是同一个内存地址,所以完全一样

第二种 浅拷贝copy

  • s1 = s2.copy()
  • 对于浅copy来说，第一层创建的是新的内存地址，
  • 而从第二层开始，指向的都是同一个内存地址，
  • 所以，对于第二层以及更深的层数来说，保持一致性。

第三种:深拷贝deepcopy

  • import copy
  • str2 =copy.deepcopy(str1)

流程控制

if的使用

if　条件：
　　　代码块
elif　条件：
　　　　代码块
else：上面条件都不满足要执行的代码

while循环 

标准格式:
while 条件:
       循环体
else:
   执行完要执行的内容

for循环 

标准格式:
for　i　in　可迭代对象：
　　　循环体
else：
　　　循环执行完毕要执行的代码

补充

  • range（）：里面有三个参数：起始位置，结束位置，步长
  • break（）：结束这个循环
  • continue（）：停止本次循环，继续下一次循环
